Besthostnews creates a dashboard widget that displays the RSS feed from www.thewebmaster.com

== Description ==

It is sometimes easy to forget to keep your pulse on the latest hosting news which may affect your site.  Keep updated with this plugin that provides the RSS feed from www.thewebmaster.com

Feature requests and ideas welcome: jonathan@thewebmaster.com

== Installation ==

1. Upload the plugin.
2. Activate it.
3. Check your dashboard.
